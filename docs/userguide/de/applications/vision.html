Vision

Deskbar:Anwendungen
Ort:/boot/system/apps/Vision/Vision
Einstellungen:~/config/settings/Vision/*


Vision ist ein IRC Client, der ursprünglich für das BeOS entwickelt wurde. Auf der Vision Webseite findet man Anleitung und Bugtracker.

In den Haiku Kanälen findet man in verschiedenen Sprachen Hilfe von anderen Benutzern und Entwicklern.
Am besten besucht ist der englischsprachige Kanal #haiku von irc.freenode.org, es existiert dort allerdings mit #haiku-de auch ein deutscher Kanal.

Hier einige Tipps, vor allem für IRC-Neulinge:

  • Such mal nach "IRC Regeln" um die grundsätzlichen Verhaltensregeln im Chat zu kennen. Die Seite IRC FAQ, Net(t)iquette, Regeln & Infos ist beispielsweise nicht schlecht.

  • Wen man seinen Nicknamen registriert, kann niemand anderer unter diesen Namen Unfug treiben. Außerdem gewinnt man dadurch das Vertrauen der Einheimischen. Siehe dazu Freenode's FAQ.

  • In Haikus 'Sounds' Einstellungen kann man einen Klang angeben, der immer ertönt wenn der eigene Nick erwähnt wird. Schließlich hat niemand immer das Chat-Fenster im Blick. Beginnt man seine Eingaben mit dem Nicknamen der Person, die man anspricht, erhöht das die Chancen wahrgenommen zu werden. Wie das Terminal beherrscht Vision die Tab-Vervollständigung von Nicknamen, d. h. beginnt man die Eingabe eines Nicknames und drückt nach einigen Buchstaben die Tabulatortaste, vervollständigt Vision den Nicknamen mit dem ersten Treffer.

Tastaturkürzel

Vision bietet einige nützliche Tastaturkürzel:

TABTab-Vervollständigung wie im Terminal: Nachdem einige Zeichen eingegeben wurden, komplettiert ein Druck der TAB Taste zum ersten passenden Nicknamen.
↑ / ↓Ist das Texteingabefeld aktiv, bewegt man sich so durch seinen kürzlich gesendeten Texte.
ALT ↑ / ↓Aktiviert das Fenster über- bzw. unter dem momentan in der "Fensterliste" ausgewählten Fenster.
SHIFT ALT ↑ / ↓Ähnlich wie ALT ↑ / ↓, aber etwas raffinierter. Anstatt das Fenster direkt über/unter dem momentanen zu aktivieren, wird hier intelligenter vorgegangen: Existieren Fenster mit einem höheren Status als das gerade aktive, wird zu denen gesprungen. Das mit dem höchsten Status wird zuerst angesprungen.
Um das zu verstehen, muss man wissen dass Vision die Fenster in der Fensterliste je nach Status unterschiedlich einfärbt. Der Status hängt davon ab, wie sich der Inhalt verändert hat, seitdem das Fenster das letzte Mal aktiv war:
  • black - kein neuer Text
  • gray - neuer Text, aber nichts aufregendes (jemand hat den Kanal betreten oder verlassen, etc.)
  • green - jemand hat etwas geschrieben
  • red - jemand gat was geschrieben und den Nicknamen des Users gesagt
Vision behandelt mit abnehmender Priorität. Wird SHIFT ALT ↑ / ↓ gedrückt, springt Vision erst zu dem nächsten roten Fenster. Gibt es kein rotes, wird zum grünen gesprungen und so weiter.

Befehle

Hier eine Liste der in Vision eingebauten Befehle, wie sie auf der Visoion Webseite zu finden sind:

Die Syntax:

/BEFEHLder Befehl
<erforderlich>ein erforderlicher Parameter
[optional]ein optionaler Parameter
|ein logisches ODER
...weitere mögliche Parameter

  • /ABOUT
    Öffnet Visions "Über" Fenster.

  • /ACRONYM [Wort]
    Öffnet www.acronymfinder.com mit dem für HTML gesetzten Programm. (Falls kein Browser dafür gesetzt wurde, wird die in den FileTypes festgelegte Anwendung geöffnet.)

  • /AWAY [Grund-der-Abwesenheit]
    Setzt den Abwesenheitsstatus im aktuellen Netzwerk. Wurde kein Grund-der-Abwesenheit angegeben, wird 'BRB' ('be right back' = ' bin gleich wieder zurück') gesetzt.

  • /BACK
    Setzt den Abwesenheitsstatus im aktuellen Netzwerk auf 'back' (= 'bin zurück').

  • /CLEAR
    Löscht den Inhalt des aktuellen Textpuffers.

  • /CTCP <Nickname|Kanal> <PING|VERSION|FINGER|TIME|CLIENTINFO|USERINFO>
    Führt den angegebenen CTCP Befehl auf einen Nicknamen bzw. Kanal aus.

  • /DCC <CHAT> <Nickname>
    Öffnet eine private DCC Chat-Sitzung mit dem angegebenen Nicknamen.

  • /DCC <SEND> <Nickname> [file]
    Schickt eine Datei an <Nickname>. Wurde keine Datei angegeben, öffnet sich ein Dateidialog.

  • /DEOP /DOP /DEVOICE <Nickname1> [Nickname2] ...
    Entzieht den Operatorstatus (De-op) oder das Rederecht (De-voice) für den/die angegebenen <Nickname(n)>.

  • /DESCRIBE <Nickname> <Nachricht>
    Wie ein privates /ME, nur <Nickname> kann es sehen.

  • /DNS <Domänenname/IP Adresse>
    Löst die angegebene IP oder Domänennamen auf.

  • /EXIT
    Beendet Vision.

  • /GAWAY [Grund-der-Abwesenheit]
    Wie /AWAY [Grund-der-Abwesenheit], aber für alle verbundenen Netzwerke.

  • /GBACK
    Wie /BACK, aber für alle verbundenen Netzwerke.

  • /GOOGLE [Suchbegriff]
    Öffnet www.google.com mit dem für HTML gesetzten Programm. (Falls kein Browser dafür gesetzt wurde, wird die in den FileTypes festgelegte Anwendung geöffnet.)

  • /INVITE /I <Nickname> <Kanal>
    Lädt <Nickname> in den <Kanal> ein.

  • /JOIN /J <Kanal> [Chiffrierschlüssel]
    Um den <Kanal> zu betreten, ggf. mit einem benötigten [Chiffrierschlüssel].

  • /KICK /K <Nickname> [Begründung]
    Schmeißt <Nickname> aus dem aktuellen Kanal. Wurde keine [Begründung] angegeben, wird die in den Einstellungen gesetzte benutzt.

  • /KILL <Nickname> [message]
    Der IRC Operatorbefehl um einen User vom Netzwerk zu trennen.

  • /LIST
    Öffnet eine Liste aller Kanäle des aktuellen Netzwerks. [außer versteckte Kanäle]

  • /M
    Zeigt die Kanalmodi des aktuellen Kanals.

  • /M <+-ohvbeqa> <Nickname1> [Nickname2] ...
    Wie /MODE <Kanal> <Modi> <Nickname>, aber für den aktuellen Kanal.

  • /M <+-kfL,l,psmntirRcOAQKVHGCuzN>
    Wie /MODE <Kanal> <Modi>, aber für den aktuellen Kanal.

  • /ME <Text>
    Zeigt: * {DeinNickname} <Text>.

  • /MODE <Kanal> <+-ohvbeqa> <Nickname1> [Nickname2] ...
    Setzt den Modus für <Nickname(n)> im <Kanal>.
    z.B. /MODE #Kanal +o Nickname
    oder /MODE #Kanal +oo-o Nickname1 Nickname2 Nickname3

  • /MODE <Kanal> <+-kfL,l,psmntirRcOAQKVHGCuzN>
    Setzt den Modus für <Kanal>. (Hinweis: Kommas dienen nur der optischen Abgrenzung.)
    z.B. /MODE #Kanal +ms

  • /MSG <Nickname> <Nachricht>
    Sendet eine <Nachricht> an <Nickname>.

  • /NAMES
    Zeigt alle Kanäle und alle Nicknamen im Netzwerk im Nezerkfenster. (Achtung, das könnte länger dauern...)

  • /NAMES <Kanal>
    Zeigt alle Nicknamen im <Kanal>.

  • /NICK <neuer-Nickname>
    Ändert den eigenen Nicknamen zu <neuer-Nickname>.

  • /NOTICE <Nickname> <Nachricht>
    Schickt eine Mitteilung an <Nickname> mit der angegebenen <Nachricht>.

  • /NOTIFY <Nickname1> [Nickname2] ...
    <Nickname(n)> werden zur Mitteilungsliste hinzugefügt.

  • /PEXEC /RRUN <Anwendung1> [ | <Anwendung2> | ... ]
    Führt die angegebene <Anwendung> aus. Werden mehrere Anwendungen angegeben, werden diese in einer Pipe ausgeführt.
    z.B. /PEXEC Terminal | StyledEdit öffnet ein neues Terminal und danach StyledEdit
    oder /PEXEC ls -la ~/Downloads/ zeigt den Inhalt des ~/Downloads/ Ordners im aktuellen Fenster. (Hier ist Vorsicht geboten!)

  • /OP /VOICE <Nickname1> [Nickname2] ...
    Operatorstatus bzw. Rederecht für <Nickname(n)>.

  • /PART
    Verlässt den aktuellen Kanal oder Nezwerk.

  • /PING <Nickname>
    Pingt <Nickname> gibt die Ping-Zeit zurück. Entspricht /CTCP <Nickname> PING.

  • /QUERY /Q <Nickname> [Nachricht]
    Öffnet ein Query-Fenster zu <Nickname>, optional mit einer [Nachricht].

  • /QUIT /QUI [spezielle-Trenn-Nachricht]
    Trennt vom aktuellen Netzwerk. Wurde keine [spezielle-Trenn-Nachricht] angegeben, wird die in den Einstellungen gesetzte benutzt.

  • /RAW /QUOTE <Befehl|Text>
    Sendet einen direkten IRC Befehl.

  • /RECONNECT
    Verbindet erneut zum aktuellen Netzwerk, falls die 47 automatischen Neu-Verbindungen fehlgeschlagen sind.

  • /SLEEP <10-Sekunden>
    Hält den aktuellen Thread für die angegebene Zeit an; für zukünftige Skriptingunterstützung.

  • /SQUIT
    IRC Operatorbefehl um einen Server zu trennen.

  • /TIME <Nickname>
    Zeigt die lokale Uhrzeit von <Nickname>. Entspricht /CTCP <Nickname> TIME.

  • /TOPIC /T [Kanal]
    Zeigt das Motto des aktuellen Kanals oder, falls angegeben, von <Kanal>.

  • /TOPIC <Kanal> <neues-Kanal-Motto>
    Setzt das Motto von <Kanal> zu <neues-Kanal-Motto>.

  • /TOPIC <neues-Kanal-Motto>
    Wie /TOPIC <Kanal> <neues-Kanal-Motto>, aber für den aktuellen Kanal.

  • /UNNOTIFY <Nickname1> [Nickname2] ...
    Entfernt <Nickname(n)> von der Mitteilungsliste.

  • /UPTIME [-l]
    Zeigt die Laufzeit des eigenen Systems. Wurde [-l] angegeben, wird sie nur lokal angezeigt und nicht zum Netzwerk gesendet.

  • /VERSION <Nickname>
    Zeigt die Version von<Nicknames> IRC-Programms. Entspricht /CTCP <Nickname> VERSION.

  • /VISIT <url>
    Öffnet <url> mit dem für HTML gesetzten Programm. (Falls kein Browser dafür gesetzt wurde, wird die in den FileTypes festgelegte Anwendung geöffnet.)

  • /WEBSTER /DICTIONARY [Wort]
    Öffnet www.m-w.com (Merriam Webster) mit dem für HTML gesetzten Programm. (Falls kein Browser dafür gesetzt wurde, wird die in den FileTypes festgelegte Anwendung geöffnet.)

  • /VUPTIME [-l]
    Zeigt die Laufzeit von Vision. Wurde [-l] angegeben, wird sie nur lokal angezeigt und nicht zum Netzwerk gesendet.

  • /WALLOPS
    Ein IRC Operatorbefehl, um eine 'Wallops' Nachricht zu schicken (nur für Leute mit aktiviertem umode w)

  • /WHOIS /W <Nickname>
    Zeigt die Informationen zu <Nickname>.
